# Original source — the classical result

Joint convexity of the Umegaki relative entropy is **classical**. The proof formalized
in this contribution uses a different (sandwiched, α→1⁺) route, so there is no single
"original paper" whose proof is reproduced — but the classical origin, against which our
route is compared in [`../../test/literature-correspondence.md`](../../test/literature-correspondence.md),
is:

## G. Lindblad (1974)

> G. Lindblad, **Expectations and entropy inequalities for finite quantum systems**,
> *Communications in Mathematical Physics* **39** (1974), no. 2, 111–119.

- DOI: [10.1007/BF01608390](https://doi.org/10.1007/BF01608390)
- Project Euclid: <https://projecteuclid.org/euclid.cmp/1103860161>
- Springer: <https://link.springer.com/article/10.1007/BF01608390>

Citation verified 2026-07-06 (author, title, journal, volume, pages, year, DOI all
confirmed against Springer and Project Euclid).

Related, for the monotonicity/data-processing direction:

> G. Lindblad, **Completely positive maps and entropy inequalities**,
> *Commun. Math. Phys.* **40** (1975), 147–151.
> DOI [10.1007/BF01609396](https://doi.org/10.1007/BF01609396).

## Why the PDF is not included here

The 1974 paper is **subscriber-access** on both Springer and Project Euclid — Project
Euclid states it "is only available to subscribers … not available for individual sale."
It is **not** open-access or openly licensed. Committing the PDF to this public
repository would be redistribution of copyrighted material Springer restricts, so it is
intentionally **omitted**; the canonical links above are provided instead.

If you have legitimate institutional access and wish to keep a local copy for your own
review, place it beside this file as `Lindblad1974.pdf` — but note that it should **not**
be pushed to a public remote. (A `.gitignore` entry in this directory guards against an
accidental commit.)
